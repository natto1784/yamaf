Cleanup

File retention is as per the following formula 

      max_size = {{MAX_SIZE}} // in megabytes
      max_days = {{MAX_DAYS}}
      min_days = {{MIN_DAYS}}
      retention = min_days + (max_days - min_days) * (1 - file_size / max_size) ^ e
Cleanup happens everyday when midnight (IST) has passed
